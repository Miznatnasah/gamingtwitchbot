# Gaming Twitch Bot

A Twitch Bot that tracks the account that a user is playing on and the record of that ranked session, and the rank of the account that the user is playing on.

## Description

A twitch chatter just has to use the commands !rank or !record in chat and the bot will respond with the current rank and the peak rank of the current account the streamer is playing on, or the current ranked session record of the current account.
